### 线性表 (Linear List)

线性表就是数据排成像一条线一样的结构，每个线性表上的数据最多只有前和后两个方向。数组，队列，链表，栈都是线性表结构。

![image](./images/b6b71ec46935130dff5c4b62cf273477.jpg)

### 非线性表

二叉树、堆、图等都是非线性，因为在非线性表中，数据之间并不是简单的前后关系。
